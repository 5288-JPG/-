在命令行/终端中，进入到包含你保存的 .py 文件的文件夹。

运行以下命令：

bash
Copy code
python merge_video_audio.py
如果你的系统使用 Python3，可能需要使用 python3 替代 python：

Copy code

执行完代码后，会在当前文件夹中生成合并后的视频文件。你可以在同一文件夹中找到输出文件，其文件名以 output_ 开头，后面跟着原始视频文件名。
